# 🛡️ CharacterGuard: AI Adversarial Testing Framework

This repository contains the CharacterGuard framework, designed to automatically test and evaluate AI characters against prompt injections, persona drift, and compliance breaks.

## 🏗️ Architecture
* **`app/`**: The Streamlit frontend dashboard and UI pages.
* **`backend/`**: The core Python engine for running attacks (Pending).
* **`evaluation/`**: The AI grading and scoring logic (Pending).
* **`data/reports/`**: The JSON output data contracts.
